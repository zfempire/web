过滤器的使用及自定义过滤器

list
{% templatetag openvariable %} list | join:"," {% templatetag closevariable %}
list = ["苹果","橘子","香蕉","西瓜"]  结果：{{ list|join:", " }}
default
{% templatetag openvariable %} value | default:"nothing" {% templatetag closevariable %}
default过滤器，为变量设置缺省值
add
{% templatetag openvariable %} value|add:"2" {% templatetag closevariable %}
将value加上参数设定的值，例如value=4 ，则输出6。 也可以用于将一个列表append到另一个列表中
date
{% templatetag openvariable %} value|date:"D d M Y" {% templatetag closevariable %}
格式化日期{{ nowtime|date:"Y m d D" }}
